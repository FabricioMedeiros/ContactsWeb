Contacts Web
Este projet é uma Aplicação Web de Agenda de Contatos.

🚀 Começando
Essas instruções permitirão que você obtenha uma cópia do projeto em operação na sua máquina local para fins de desenvolvimento e teste.

Consulte Implantação para saber como implantar o projeto.

🛠️ Construído com
Visual Studio
SQL Server - Gerenciador de Banco de Dados
Entity FrameWork - Framework Data Base utilizado
ASPNET CORE - Framework Web utilizado
Bootstrap - Framework para estilização dos formulários
CSS - para estilização
Java Script/Jquery
